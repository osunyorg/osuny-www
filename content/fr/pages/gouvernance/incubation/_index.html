Définir la vision

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 39 text: >- 1 minute contents: - kind: block template: title title: >- 2021-2022 slug: >- 2021-2022 ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

La première phase est portée par noesya, sur ses fonds propres. Elle a commencé en septembre 2021, et devrait s'achever en août 2022. Tous les résultats de cette première phase sont partagés de façon ouverte sur le site osuny.org.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Vision stratégique slug: >- vision-strategique ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Le premier enjeu est de proposer une vision stratégique claire d'Osuny, afin de positionner la plateforme au sein des écosystèmes préexistants (enseignement supérieur, recherche, systèmes d'information, open source...). 

notes: >- alt: >- credit: >- - kind: block template: title title: >- Démonstrateur opérationnel slug: >- demonstrateur-operationnel ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Le second enjeu est la réalisation de preuves de concepts (Proof Of Concept, POC), en production et en préproduction. 

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cœur du réseau de parties prenantes slug: >- coeur-du-reseau-de-parties-prenantes ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Le troisième enjeu, corrélatif au second, est de constituer un noyau de parties prenantes, afin d'ancrer le produit dans une réalité tangible et de préparer les étapes ultérieures.
