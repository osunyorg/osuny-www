Thème Figma
https://www.figma.com/community/file/1176490763864113451/Th%C3%A8me-Osuny


Le thème Figma contient tout le design system d'Osuny, les typos, les couleurs, les espaces, les menus, la navigation, la pagination, les icônes, etc. Il présente aussi toutes les configurations possibles de blocs, et les particularités des sections (actualités, formations...).
