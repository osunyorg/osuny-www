Que se passe-t-il si noesya fait faillite ? Et si toute l'équipe passe sous un camion ? Si l'on ne veut plus utiliser Osuny ? Est-on bloqués ? À qui appartiennent les données ? À qui appartient le code source du site ? À qui appartient le code source d'Osuny ? 

notes: >- alt: >- credit: >- - kind: heading title: >- La propriété slug: "la-propriete" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Le code source des sites, comme le code source du thème et de l'admin, sont des logiciels libres. Cela signifie qu'ils sont utilisables librement par qui veut. Ils ne sont pas la propriété de noesya, ils appartiennent à tout le monde. La licence choisie est la licence MIT, qui est permissive. Cela signifie que vous pouvez utiliser le code source comme vous voulez, même dans un cadre privatif (code source non libre). Nous avons fait ce choix parce que nous pensons qu'il est plus important de réparer les sites Web afin qu'ils soient accessibles et sobres, plutôt que de protéger le libre à tout prix. Concrètement, si une entreprise utilise le thème Osuny pour fabriquer un produit privatif sur abonnement, la licence le permet. Nous ne sauterons pas de joie, mais la licence l'autorise, et on espère que cela aboutira à des sites plus sains.
