Gestion des comptes utilisateurs sécurisée avec authentification multi-facteur (MFA) et conforme au Règlement Général sur la Protection des Données (RGPD). Ergonomie simple, proche de WordPress et intégrant les logiques spécifiques aux Universités et aux écoles.

notes: >- alt: >- credit: >- - kind: heading title: >- Génération de sites précompilés slug: "generation-de-sites-precompiles" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Publication automatique des données sur un référentiel Github (Open Source par défaut, mais le référentiel privé est techniquement possible). Développement avec Hugo du site, personnalisation du style, optimisation de performance et d'accessibilité.

notes: >- alt: >- credit: >- - kind: heading title: >- Migration automatique depuis WordPress slug: "migration-automatique-depuis-wordpress" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Afin de faciliter la gestion du changement, Osuny intègre un système d'import depuis WordPress : articles, pages, catégories et auteurs. Les contenus textuels sont importés et nettoyés automatiquement, les visuels sont extraits et stockés pour réutilisation et optimisation dans le site précompilé.

notes: >- alt: >- credit: >- - kind: heading title: >- Hébergement souverain slug: "hebergement-souverain" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

La coopérative noesya propose un hébergement commun, avec un silo applicatif par Université. Cet hébergement utilise les services français Scalingo et Scaleway. Le site précompilé peut-être mis en ligne sur l'hébergement que vous souhaitez, avec un flux de publication appuyé sur Git : Netlify, serveurs virtuels privés, serveurs dédiés...
