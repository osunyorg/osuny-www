Le back-office codé avec Ruby on Rails est en licence MIT1, ainsi que le thème Hugo2, le template pour faciliter l'utilisation du thème3, le site d'exemple du thème4, la documentation pour les utilisatrices et utilisateurs5, la documentation pour les développeuses et développeurs6.

Les technologies utilisées pour construire Osuny sont elles aussi libres, à l'exception du thème graphique utilisé pour l'admin (Appstack), pour lequel noesya a acquis une licence d'utilisation compatible avec un projet libre. Le back-office est construit avec Ruby1 (Licence Ruby2), Ruby on Rails3 (Licence MIT4), PostgreSQL5 (licence PostgreSQL6), et un ensemble de gems consultables sur le référentiel Github du projet7. Le front-office est construit avec Hugo8 (Licence Apache 2.09) et le moins possible de dépendances, listées sur le référentiel Github du thème10.
