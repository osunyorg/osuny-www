L'accessibilité, c'est tout simplement le numérique au service de toutes et tous. Le thème Hugo qui sert de base aux sites réalisés avec Osuny tente d'assurer une conformité parfaite avec le RGAA. La pile technologique utilisée pour le back-office propose un équilibre entre maintenabilité et accessibilité : Bootstrap 5, thème d'administration AppStack, adaptations spécifiques.

Les exigences d’Osuny sur la qualité et l’accessibilité nous font gagner un temps précieux pour remplir nos obligations dans ces domaines

Le front-office est particulièrement rétro-compatible (testé et fonctionnel jusqu'à Windows Vista, 2007), ce qui évite d'exclure les personnes utilisant de vieux périphériques. Le back-office est moins rétro-compatible, mais il fonctionne néanmoins avec des devices anciens.

Les sites sont particulièrement légers en transferts de données, ce qui évite d'exclure les personnes ayant de faibles connexion au réseau. Le back-office est lui aussi optimisé pour minimiser les transferts de données.
