Osuny est un logiciel libre. Comme d'autres logiciels (WordPress, K-Sup, Ametys...), son code est ouvert et gratuit, vous pouvez l'utiliser comme vous le souhaitez dans le cadre de la licence MIT. 

notes: >- alt: >- credit: >- - kind: heading title: >- Payer les prestations de développement ad hoc slug: "payer-les-prestations-de-developpement-ad-hoc" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez solliciter n'importe quel prestataire pour produire un site appuyé sur Osuny, cela n'a aucun impact sur le prix de développement. Chaque prestataire devise son travail en fonction de ses propres choix, tant pour le développement initial que pour la maintenance du site (TMA).
