Comprendre les coûts initiaux et récurrents de la solution

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 43 text: >- 1 minute contents: - kind: block template: title title: >- Utiliser le code gratuitement slug: >- utiliser-le-code-gratuitement ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Osuny est un logiciel libre. Comme d'autres logiciels (WordPress, K-Sup, Ametys...), son code est ouvert et gratuit, vous pouvez l'utiliser comme vous le souhaitez dans le cadre de la licence MIT. Il manque de la documentation pour le déploiement de nouveaux serveurs Osuny, noesya se concentre sur Osuny as a Service, qui sera le socle de la future SCIC Osuny.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Payer les prestations de développement ad hoc slug: >- payer-les-prestations-de-developpement-ad-hoc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous pouvez solliciter n'importe quel prestataire pour produire un site appuyé sur Osuny, cela n'a aucun impact sur le prix de développement. Chaque prestataire devise son travail en fonction de ses propres choix, tant pour le développement initial que pour la maintenance du site (TMA).

noesya propose un hébergement souverain d'Osuny avec Scalingo et Scaleway, sans limite de nombre de personnes utilisatrices, intégrant la maintenance du back-office. Cette offre est détaillée dans la page Osuny as a Service.
