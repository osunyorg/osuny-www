* content management system, système de gestion de contenus

body: >-

Osuny est un système de gestion de contenus libre qui permet de créer des sites accessibles, éco-conçus, qualitatifs, souverains et sécurisés.
Utiliser une licence libre

Osuny est un logiciel libre, en licence MIT. Le back-office codé avec Ruby on Rails est en licence MIT, ainsi que le thème Hugo, le template pour faciliter l'utilisation du thème, le site d'exemple du thème, la documentation pour les utilisatrices et utilisateurs, la documentation pour les développeuses et développeurs.

Un outil sobre

Les technologies utilisées pour construire Osuny sont elles aussi libres, à l'exception du thème graphique utilisé pour l'admin (Appstack), pour lequel noesya a acquis une licence d'utilisation compatible avec un projet libre. Le back-office est construit avec Ruby (Licence Ruby2), Ruby on Rails3
(Licence MIT4), PostgreSQL5 (licence PostgreSQL6), et un ensemble de gems consultables sur le référentiel Github du projet. Le front-office est construit avec Hugo (Licence Apache 2.09) et le moins possible de dépendances, listées sur le référentiel Github du thème10.

Un outil accessible

L'accessibilité, c'est tout simplement le numérique au service de toutes et tous. Le thème Hugo qui sert de base aux sites réalisés avec Osuny tente d'assurer une conformité parfaite avec le RGAA. La pile technologique utilisée pour le back-office propose un équilibre entre maintenabilité et accessibilité: Bootstrap 5, thème d'administration
AppStack, adaptations spécifiques.
